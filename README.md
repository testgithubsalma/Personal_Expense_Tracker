# 💸 Personal Expense Tracker

A simple and interactive **Personal Expense Tracker Web Application** built using **Python, Pandas, Matplotlib, and Streamlit**.

## 🚀 Live Demo

🔗 [https://personal-expense-tracker-appqj3i3znjqpzhcf4fnwpw.streamlit.app/](https://personalexpensetracker-8vjvnkucnnybjg4jcui3qv.streamlit.app/)

---

# 📌 Features

✅ Add Income Transactions
✅ Add Expense Transactions
✅ View Complete Transaction History
✅ Download Transactions as CSV
✅ Financial Summary Dashboard
✅ Expense Bar Chart Visualization
✅ Expense Pie Chart Visualization
✅ User-Friendly Streamlit Interface

# ▶️ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/testgithubsalma/Personal_Expense_Tracker.git
```

## 2️⃣ Open Project Folder

```bash
cd Personal_Expense_Tracker
```

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

## 4️⃣ Run Streamlit App

```bash
streamlit run app.py
```
